Why bd-geo-info

Everything you need, nothing you don't

Built to work anywhere JavaScript runs — frontend, backend, serverless, or edge.

🔌

Framework Agnostic

Core utilities have zero runtime dependencies. Works with React, Vue, Angular, Svelte, NestJS, Express, or plain Node.js.

⚡

O(1) Lookups

Datasets indexed by parent ID — no linear scans across thousands of records. Instant results at any depth.

📦

Code Split

Large JSON datasets use dynamic import() and load on demand — never bloat your initial bundle.

🌐

Bilingual

Every entry has both English and বাংলা names. Switch output language with a single parameter.

🔷

Full TypeScript

Strict types for all data shapes and utilities. Separate types for core and React so non-React projects stay clean.

⚛️

Optional React UI

Pre-built cascading selects via bd-geo-info/react. Non-React users pay zero cost.

🗜️

Minified Data

All JSON assets minified at build time — ~300 KB saved vs raw source on every install.

🌲

Tree Shakeable

Declares "sideEffects": false. Bundlers drop every export you don't use.

Quick Start

index.ts 
import { getDivisions, getDistricts, getUpazilas, getUnions } from 'bd-geo-info';

// All divisions (8 divisions of Bangladesh)
const divisions = await getDivisions('en');
// [{ value: '1', label: 'Chittagong' }, { value: '3', label: 'Dhaka' }, ...]

// Districts for a division
const districts = await getDistricts('3', 'en');

// Upazilas for a district
const upazilas = await getUpazilas('17', 'en');

// Unions for an upazila
const unions = await getUnions('101', 'en');

// বাংলা output — just change the language param
const divisionsBn = await getDivisions('bn');
// [{ value: '1', label: 'চট্টগ্রাম' }, ...]
API Reference

All utility functions

Import from bd-geo-info — no framework required.

getDivisions async
getDivisions(language?: 'en' | 'bn'): Promise<{ value: string; label: string }[]>

Returns all 8 divisions of Bangladesh. Defaults to English.

const divisions = await getDivisions('en');
// [{ value: '1', label: 'Chittagong' }, { value: '3', label: 'Dhaka' }, ...]
const divisionsBn = await getDivisions('bn');
// [{ value: '1', label: 'চট্টগ্রাম' }, ...]
getDistricts async
getDistricts(divisionId: string, language?: 'en' | 'bn'): Promise<{ value: string; label: string }[]>

Returns all districts belonging to a division.

const districts = await getDistricts('3', 'en');
// [{ value: '17', label: 'Dhaka' }, { value: '26', label: 'Gazipur' }, ...]
getUpazilas async
getUpazilas(districtId: string, language?: 'en' | 'bn'): Promise<(Upazila & { value: string; label: string })[]>

Returns all upazilas for a district. Includes full data fields alongside value / label.

const upazilas = await getUpazilas('17', 'en');
// [{ id: '101', district_id: '17', name: 'Mirpur', value: '101', label: 'Mirpur' }, ...]
getUnions async
getUnions(upazilaId: string, language?: 'en' | 'bn'): Promise<{ value: string; label: string; url?: string }[]>

Returns all unions for an upazila.

const unions = await getUnions('101', 'en');
// [{ value: '1001', label: 'Pallabi', url: '' }, ...]
getPostCode sync
getPostCode(params: { division?: string; district?: string; upazila?: string } | null): PostCode[]

Synchronous postcode lookup. Useful inline without await.

const codes = getPostCode({ district: '17', upazila: 'Mirpur' });
// [{ postCode: '1216', postOffice: 'Pallabi', upazila: 'Mirpur', ... }]
formatAddress async
formatAddress(division, district, upazila, union, postCode, street, language?): Promise<string>

Resolves IDs to names and returns a human-readable formatted address string.

const addr = await formatAddress('3', '17', '101', '1001', '1216', 'House 5', 'en');
// "House 5, Pallabi, Mirpur, Dhaka-1216, Dhaka"
validatePostCode sync
validatePostCode(postCode: string): boolean

Returns true for a valid 4-digit Bangladesh postcode.

validatePostCode('1216'); // true
validatePostCode('12');   // false
validatePostCode('abcd'); // false
Framework Guides

Works everywhere

The same three imports, any stack.

import { getDivisions, getDistricts, getUpazilas, getUnions, getPostCode } from 'bd-geo-info';

const divisions = await getDivisions('en');
const districts = await getDistricts('3', 'en');
const upazilas  = await getUpazilas('17', 'en');
const unions    = await getUnions('101', 'en');
const codes     = getPostCode({ district: '17', upazila: 'Mirpur' });
import { useState, useEffect } from 'react';
import { getDivisions, getDistricts } from 'bd-geo-info';

export function AddressSelector() {
  const [divisions, setDivisions] = useState([]);
  const [districts, setDistricts] = useState([]);
  const [divisionId, setDivisionId] = useState('');

  useEffect(() => { getDivisions('en').then(setDivisions); }, []);
  useEffect(() => {
    if (divisionId) getDistricts(divisionId, 'en').then(setDistricts);
    else setDistricts([]);
  }, [divisionId]);

  return (
    <select value={divisionId} onChange={e => setDivisionId(e.target.value)}>
      {divisions.map(d => <option key={d.value} value={d.value}>{d.label}</option>)}
    </select>
  );
}
<script setup lang="ts">
import { ref, watch, onMounted } from 'vue';
import { getDivisions, getDistricts } from 'bd-geo-info';

const divisions = ref([]);
const districts = ref([]);
const selectedDivision = ref('');

onMounted(async () => { divisions.value = await getDivisions('en'); });

watch(selectedDivision, async (id) => {
  districts.value = id ? await getDistricts(id, 'en') : [];
});
</script>

<template>
  <select v-model="selectedDivision">
    <option v-for="d in divisions" :key="d.value" :value="d.value">{{ d.label }}</option>
  </select>
</template>

import { Injectable } from '@angular/core';
import { from } from 'rxjs';
import { getDivisions, getDistricts, getUpazilas } from 'bd-geo-info';

@Injectable({ providedIn: 'root' })
export class GeoService {
  getDivisions(lang = 'en') { return from(getDivisions(lang)); }
  getDistricts(id: string, lang = 'en') { return from(getDistricts(id, lang)); }
  getUpazilas(id: string, lang = 'en') { return from(getUpazilas(id, lang)); }
}
<script lang="ts">
  import { onMount } from 'svelte';
  import { getDivisions, getDistricts } from 'bd-geo-info';

  let divisions = [], districts = [], divisionId = '';

  onMount(async () => { divisions = await getDivisions('en'); });

  async function onDivisionChange() {
    districts = divisionId ? await getDistricts(divisionId, 'en') : [];
  }
</script>

<select bind:value={divisionId} on:change={onDivisionChange}>
  {#each divisions as d}<option value={d.value}>{d.label}</option>{/each}
</select>
import { Injectable } from '@nestjs/common';
import { getDivisions, getDistricts, getUpazilas, getUnions } from 'bd-geo-info';

@Injectable()
export class GeoService {
  getDivisions(lang: 'en' | 'bn' = 'en') { return getDivisions(lang); }
  getDistricts(id: string, lang = 'en') { return getDistricts(id, lang); }
  getUpazilas(id: string, lang = 'en') { return getUpazilas(id, lang); }
  getUnions(id: string, lang = 'en') { return getUnions(id, lang); }
}
import express from 'express';
import { getDivisions, getDistricts, getUpazilas } from 'bd-geo-info';

const app = express();

app.get('/api/divisions', async (req, res) => {
  res.json(await getDivisions((req.query.lang ?? 'en') as 'en' | 'bn'));
});

app.get('/api/districts', async (req, res) => {
  const { divisionId, lang = 'en' } = req.query as any;
  res.json(await getDistricts(divisionId, lang));
});

app.listen(3000);
